Programming with Professors

Dear SNU folks!

We have been working on something called "Programming with Professors". We have been asking faculty if they could give us they had problems in research/teaching which could be solved via programming. We would then compile a list of problem statements which interested chapter members can pick from and solve, improving their programming skill.Example: a student who knows R/Python could write scripts that aid in data analysis/implement an algorithm/simulate a phenomena, whose output would be useful to the professor in his/her work."

We have five people on board: Santosh Kumar, Priya Johari, Partha Chatterjee, Prasad Pathak, and Ganesh Prabhu.

1. Dr. Santosh Kumar Physics Interested in: Computational Physics problems.

2. Dr. Priya Johari Physics Interested in: Computational solutions to specific problems. See problem set.

3. Dr. Partha Chatterjee Economics Interested in: Modelling in Econometrics (Monte Carlo simulations, Dynamic programming). An app for surveying/recording data (data collection). Data visualization. Scripting to automate handling of large quantities of data. See notes.

4. Dr. Prasad Pathak Civil Engineering Interested in: Mapping. Using the APIs and scripts to work with map data.

5. Ganesh Prabhu (Research Scholar) Center for Informatics Interested in: Competence in Python/R/Shell for mining chemical information datasets.
